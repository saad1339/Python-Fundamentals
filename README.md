# Python Programming Lab Tasks

This repository contains my Python programming lab tasks completed as part of my coursework. These exercises cover the fundamental concepts of Python programming, including input/output, variables, arithmetic operations, operators, and basic problem-solving.

## 📚 Topics Covered

* Printing formatted output
* User input
* Variables and data types
* ASCII value using `ord()`
* Arithmetic operations
* Swapping two numbers without using a third variable
* Bus seat calculation
* Electricity bill calculation

## 📂 Tasks

### Task 1 – Student Information

Prints the university name, student name, roll number, and class in a box format.

### Task 2 – Full Name

Accepts the user's first and last name and displays them together.

### Task 3 – ASCII Value

Takes a character as input and prints its ASCII value using the `ord()` function.

### Task 4 – Arithmetic Operations

Accepts three numbers and calculates:

* Sum
* Multiplication
* Division

### Task 5 – Swap Two Numbers

Swaps two numbers without using a third variable.

### Task 6 – Bus Seat Calculator

Calculates the number of empty seats in the last bus, assuming each bus has a capacity of **50 passengers**.

### Task 7 – Electricity Bill Calculator

Calculates the electricity bill based on units consumed in different categories.

| Category        | Rate        |
| --------------- | ----------- |
| First 100 units | 10 PKR/unit |
| Next 100 units  | 15 PKR/unit |
| Above 200 units | 20 PKR/unit |
